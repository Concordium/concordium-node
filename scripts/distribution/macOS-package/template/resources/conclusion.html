Running and stopping Concordium Node

Four shortcuts have been installed in your /Applications folder for manually controlling the node.
If you chose to run nodes on startup, these shortcuts might not be relevant for you.

The shortcuts are:

  • Concordium Node Start Mainnet
  • Concordium Node Stop Mainnet
  • Concordium Node Start Testnet
  • Concordium Node Stop Testnet
